# Customer-churn-prediction

Customer Churn Prediction web application developed using Python, TensorFlow, Scikit-learn, Pandas, NumPy, and Streamlit. The project uses an Artificial Neural Network (ANN) model to analyze customer data and predict the probability of customer churn. Implemented data preprocessing techniques including Label Encoding, One-Hot Encoding, and Standard Scaling to improve model performance and prediction accuracy.

The application provides an interactive Streamlit interface where users can input customer details such as geography, gender, age, balance, credit score, salary, tenure, number of products, and account activity status. Based on the input, the trained deep learning model predicts whether the customer is likely to churn or stay with the bank.
